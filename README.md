# AROMA-Cross-Functional-Revenue-Operations-Transformation
End-to-end Revenue Operations system standardizing the Lead-to-Cash process across Marketing, Telesales, Sales, Training &amp; Accounting. Covers lead routing, UTM attribution, lead scoring, data recycling, a data-driven sales-comp engine, and cross-functional BI. Lifted appointment rate to ~20–30% and close rate from 36% to 50–67%.
